# 적 의도(Intent) 아이콘 생성 프롬프트

각 프롬프트를 그대로 복사해서 사용하면 됩니다.
파일명 규칙: `intent_{type}.webp`
저장 경로: `frontend/src/assets/images/GUI/`
크기: 128x128px (정사각형, 흰색 배경)

기존 아이콘 참조:
- `intent_physical.webp` — 교차된 두 자루 검, 어둡고 거친 픽셀아트
- `intent_special.webp` — 바이오해저드 심볼 + 녹색 독물, 석재 프레임
