# 웨이스트랜드 런 몬스터 & 엘리트 & 보스 설계서

> ★ = 기존 구현 (스프라이트 있음) | ☆ = 신규 (스프라이트 미제작)
> STS 조우 가중치 시스템 도입. 챕터당 보스 3종 중 1종 랜덤 선택.

---

## 조우 시스템 규칙

### 가중치 기반 선별
- 각 적 조합마다 **조우 가중치(weight)** 부여, 수치 높을수록 등장 확률 증가
- 등장 확률 = (해당 조합 가중치) / (풀 내 전체 가중치 합)

### 단계별 풀 분리 (STS 방식)
| 챕터 | 첫 N회 | 이후 |
|------|--------|------|
| 1막 | 첫 **3회**: 약한 적 풀 | 강한 적 풀 |
| 2막 | 첫 **2회**: 약한 적 풀 | 강한 적 풀 |
| 3막 | 첫 **2회**: 약한 적 풀 | 강한 적 풀 |

### 연속 등장 방지
- 직전 전투에서 등장한 조합은 **다음 전투에서 등장 불가**

### 엘리트/보스 선택
- **엘리트**: 챕터별 3종 중 1종 랜덤
- **보스**: 챕터별 3종 중 1종 랜덤 (런 시작 시 결정)

| 역할 | 설명 | 대표 몬스터 |

| **탱커** | 높은 방어, 낮은 공격 | 슬라임, 골렘, 크롤러 |
| **서포터** | 아군 버프/힐 | 감시 로봇, 형광 이끼 |
| **디버퍼** | 취약/약화/화상 부여 | 거미, 들개, 감시자 |

| **타이머** | 시간 경과 시 강화 | X-7(광폭화), 프로토타입(2페이즈) |

### 역할 조합 원칙
- **딜러 + 탱커**: 탱커 방어 깨면서 딜러 제거 우선
- **서포터 + 아무나**: 서포터 우선 제거 (힐/버프 차단)
- **디버퍼 + 딜러**: 디버프 후 딜러의 폭딜 주의
- **스워머 다수**: 광역 카드 필요, 없으면 어려움
