# 변이 단계 (Mutation Stage) 시스템 설계

> STS의 승천(Ascension) 시스템에 대응하는 난이도 단계 시스템.
> "방사능 변이가 황무지를 더 깊이 오염시켰다" 라는 테마.

---

## 기본 규칙

- **총 20단계**, 효과는 **누적 적용** (변이 5단계 = 1~5단계 효과 전부 적용)
- **해금 조건**: 해당 단계에서 3막 보스 클리어 시 다음 단계 해금
- **진행도 저장**: 로컬 세이브에 최고 해금 단계 기록
- **런 시작 시 선택**: 새 게임 시작 화면에서 0~해금된 최고 단계 중 선택
- **UI 표시**: HUD에 현재 변이 단계 아이콘 + 숫자 표시

---

## 단계별 효과

### 초급 변이 (1~5단계) — 적 강화

| 단계 | 이름 | 효과 | 설계 의도 |
|------|------|------|----------|
| 1 | 변이 징후 | 엘리트 적 출현 확률 +10% | 엘리트 조우가 늘어나 루트 선택의 중요성 증가 |
| 2 | 강화된 외피 | 일반 적 체력 +10% | 전투가 약간 길어짐 |
| 3 | 오염된 보급로 | 전투 보상 보급품 드롭률 절반 | 보급품 수급이 어려워짐 |
| 4 | 변이 근육 | 엘리트 적 체력 +15% | 엘리트전 난이도 상승 |
| 5 | 방사능 후유증 | 시작 최대 체력 -7 (84→77) | 체력 관리 압박 시작 |

### 중급 변이 (6~10단계) — 자원 압박

| 단계 | 이름 | 효과 | 설계 의도 |
|------|------|------|----------|
| 6 | 오염된 수원 | 휴식 체력 회복량 -25% | 휴식의 가치 하락, 강화/제거 선택 유도 |
| 7 | 변이 핵심부 | 보스 체력 +15% | 보스전 장기화 |
| 8 | 독성 강화 | 일반 적 공격력 +1 | 방어의 중요성 증가 |
| 9 | 불리한 운명 | 이벤트 부정적 확률 +15% (실패 확률 상승) | 이벤트 리스크 증가 |
| 10 | 변이 오염 | 시작 시 덱에 [화상] 카드 1장 추가 | 덱 정화/관리 부담 |

### 고급 변이 (11~15단계) — 경제 + 시스템 제한

| 단계 | 이름 | 효과 | 설계 의도 |
|------|------|------|----------|
| 11 | 물가 폭등 | 상점 모든 가격 +25% | 골드 효율 하락 |
| 12 | 변이 군주 | 보스 공격력 +1 | 보스전 압박 증가 |
| 13 | 보급 차단 | 보급품 소지 한도 -1 (3→2, 유물 보너스는 유지) | 보급품 운용의 전략성 증가 |
| 14 | 변이 확산 | 일반 적 체력 추가 +10% (2단계와 누적 = +20%) | 일반전도 확실한 위협 |
| 15 | 빈손의 출발 | 시작 골드 0 (기본 0이면 영향 없음, 시작 이벤트 골드 선택 가치 상승) | 초반 경제 압박 |

### 극한 변이 (16~20단계) — 극한 도전

| 단계 | 이름 | 효과 | 설계 의도 |
|------|------|------|----------|
| 16 | 변이 광전사 | 엘리트 적 공격력 +2 | 엘리트가 치명적 위협으로 |
| 17 | 연쇄 변이 | 일반 적 공격력 추가 +1 (8단계와 누적 = +2) | 일반전에서도 큰 피해 |
| 18 | 변이 진화 | 엘리트 적 체력 추가 +15% (4단계와 누적 = +30%) | 엘리트전 극한 |
| 19 | 최종 변이체 | 보스 체력 추가 +15% (7단계와 누적 = +30%), 보스 공격력 추가 +1 (누적 +2) | 보스전 극한 |
| 20 | 완전한 변이 | 각 챕터 보스 처치 후 **정예 전투 1회 추가** | 최고 난이도의 상징 |

## 구현 포인트

### 데이터 저장
- `settings.json`에 `maxMutationUnlocked: number` (0~20) 저장
- `useRunStore`에 `mutationStage: number` 상태 추가 (현재 런의 변이 단계)
- 세이브/로드에 `mutationStage` 포함

### 적용 위치
| 효과 | 적용 파일 |
|------|----------|
| 적 체력/공격력 보정 | `enemies.ts` → `createEnemy()` 또는 `enemySlice.ts` → `spawnEnemies()` |
| 시작 체력 | `MainMenuView.tsx` → `handleNewGame()` |
| 시작 덱 오염 | `MainMenuView.tsx` → 덱 초기화 직후 |
| 휴식 회복량 | `RestView.tsx` → 회복 계산 |
| 상점 가격 | `ShopView.tsx` → 가격 계산 |
| 보급품 한도 | `supplyEffects.ts` → `getMaxSupplySlots()` |
| 보급품 드롭률 | `VictoryRewardPanel.tsx` → 드롭 확률 |
| 이벤트 확률 | `events.ts` → 확률 관련 onSelect |
| 엘리트 출현율 | `mapGenerator.ts` 또는 맵 노드 배치 |
| 보스 후 추가전 | `BattleView.tsx` → 보스 승리 후 처리 |

### 변이 단계 유틸 함수 (`logic/mutationModifiers.ts`)
```typescript
export function getMutationModifiers(stage: number) {
  return {
    normalHpMult: 1 + (stage >= 2 ? 0.1 : 0) + (stage >= 14 ? 0.1 : 0),
    normalAtkBonus: (stage >= 8 ? 1 : 0) + (stage >= 17 ? 1 : 0),
    eliteHpMult: 1 + (stage >= 4 ? 0.15 : 0) + (stage >= 18 ? 0.15 : 0),
    eliteAtkBonus: stage >= 16 ? 2 : 0,
    eliteSpawnBonus: stage >= 1 ? 0.1 : 0,
    bossHpMult: 1 + (stage >= 7 ? 0.15 : 0) + (stage >= 19 ? 0.15 : 0),
    bossAtkBonus: (stage >= 12 ? 1 : 0) + (stage >= 19 ? 1 : 0),
    startMaxHpReduction: stage >= 5 ? 7 : 0,
    restHealReduction: stage >= 6 ? 0.25 : 0,
    shopPriceIncrease: stage >= 11 ? 0.25 : 0,
    supplySlotReduction: stage >= 13 ? 1 : 0,
    supplyDropHalved: stage >= 3,
    startGoldZero: stage >= 15,
    startWithBurn: stage >= 10,
    eventBadLuckBonus: stage >= 9 ? 0.15 : 0,
    bossExtraElite: stage >= 20,
  };
}
```

### UI
- **새 게임 화면**: 변이 단계 선택 슬라이더 (0~해금 단계)
  - 각 단계 호버 시 해당 단계 효과 설명 툴팁
  - 해금되지 않은 단계는 자물쇠 아이콘
- **HUD**: 변이 단계 > 0이면 좌상단에 ☢️ + 숫자 표시
- **클리어 시**: "변이 N단계 클리어!" 축하 연출 + 다음 단계 해금 알림

## 밸런스 참고

| 비교 | STS 승천 | 변이 단계 |
|------|---------|----------|
| 적 체력 강화 | 2, 7, 12, 17, 18, 19 (6단계) | 2, 4, 7, 14, 18, 19 (6단계) |
| 적 공격력 강화 | 8, 13, 16, 17, 19 (5단계) | 8, 12, 16, 17, 19 (5단계) |
| 자원 압박 | 6, 11, 15 (3단계) | 3, 6, 11, 13, 15 (5단계) |
| 시작 불이익 | 5, 10, 14 (3단계) | 5, 10, 15 (3단계) |
| 최종 도전 | 20 (심장전) | 20 (보스 후 추가 정예전) |

보급품 시스템이 추가되었으므로 STS 대비 자원 압박 단계가 2개 더 많습니다 (3단계 드롭률, 13단계 소지 한도).
