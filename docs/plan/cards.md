# 웨이스트랜드 런 전체 카드 리스트 (75종)

> 물리 공격 = 근접/육탄전, 특수 공격 = 총기/화기
> 전 챕터 공통 풀. 등급(일반/특별/희귀)으로만 희귀도 관리.
> 상태이상 카드(화상, 방사능 오염)는 별도.

타격×4, 수비×2, 보호×2, 녹슨 권총×1, 잔해 뒤지기×1

## 아키타입별 핵심 카드

### 물공덱 (물리 DPS)
청테이프 공학(파워) → 타격/연타/폭주(스케일링) → 대형 오함마(취약) → 분쇄(취약 연계) → 전기톱(다타 폭딜)

### 특공덱 (총기 DPS)
잔해 뒤지기/전리품 수색(탄약) → 속사/관통탄(기본 사격) → 과충전 코일건(올인) → 아크 캐논/네이팜(피니셔)

### 물방덱 (물리 탱크)
요새화(파워) → 수비/바리케이드/반격 태세(반사) → 방패로 밀치기(방어→공격) → 철벽(비상)

### 특방덱 (특수 탱크)
전류 흡수망/에너지 전환(방어→탄약) → 납 코팅 망토(AP 보너스) → 비상용 해독 주사/완전 정화(정화)

### 순환덱 (드로우/사이클)
경계 태세/피의 대가(0코스트 드로우) → 사각지대 찌르기/잔해 은폐(공방+드로우) → 기업 기밀 문서(폭드로우)

### 소멸덱 (소멸 시너지)
불사조의 재(파워) → 희생/무기 개조(소멸 트리거) → 철벽/완전 정화/코일건(강력 소멸) → 기업 기밀 문서(소멸 드로우)

### 광전사덱 (자해/저HP)
광기(파워) → 불법 전투 자극제/피의 대가(자해+드로우) → 광전사의 일격(잃은HP 피해) → 분노(피격→방어)

## 키워드 사전

| **취약** | 받는 물리 피해 50% 증가 (1~2턴) |

| **지속(Power)** | 전투 종료까지 유지되는 영구 효과 |
