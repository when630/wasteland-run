# 웨이스트랜드 런 (Wasteland Run) 개발 진행 현황 및 향후 계획

마일스톤 문서(`milestone.md`)를 기준으로 지금까지 달성한 기능 구현 현황과 앞으로 남은 미구현 UI 시스템의 구현 계획을 정리한 문서입니다.

## 1. 지금까지 완료된 주요 작업 내역 (Done)

### ⚔️ 전투 시스템 고도화
*   **물리/특수 공격 분리 (Damage Types)**
    *   적의 공격 패턴(Intent)에 `PHYSICAL`, `SPECIAL`, `PIERCING` 속성 추가.
    *   플레이어의 대응 방어구인 방어막(Shield, 🛡️)과 저항(Resist, 💠) 분기 처리 및 UI 아이콘 반영 완료.
*   **상태이상 (Status Effects)**
    *   화상(Burn), 맹독(Poison), 취약(Vulnerable), 약화(Weak) 시스템 도입.
    *   매 턴 피해를 주거나 공격력을 증감시키는 로직을 `useBattleStore`에 구현.
    *   Pixi.js 애니메이션(`AnimatedEnemy.tsx`)을 통해 적 머리 위에 이모지와 중첩 수(`🔥2`, `💔1`) 표출 성공.

### 🗺️ 노드 및 이벤트 확장
*   **신규 무작위 이벤트 추가 (`events.ts`)**
    *   기존 (방사능 웅덩이, 기계공 모닥불, 트럭 함정, 오락기) 4종 외 3종 추가.
    *   *폐허가 된 진료소, 고철 수집상의 시체, 기계 사이보그의 내기.*
    *   선택지에 따른 보상(골드, 유물, 체력 힐링) 및 패널티(상태이상 카드 덱 추가, 체력 감소 등)가 `useRunStore`와 자동 연동됨.
*   **상점 및 보스전 연결**
    *   `MapView`에서 잠겨있던 상점(Shop) 노드 진입 해금 완료.
    *   보스 노드 도달 시 `baseId: 'brutus'`가 스폰되는 보스 씬 자동 진입 연결 완료.

### 🏠 메인 시스템 및 세이브
*   **메인 메뉴 화면 (`MainMenuView.tsx`)**
    *   게임 최초 접속 시 임시 맵이 아닌 타이틀 로고와 메인 메뉴가 보이게 수정.
    *   **[새 게임]**: 모든 상태, 덱(기본 무기 3/2 비율), 골드, 맵 층수(1층), 백엔드 서버 기록을 `isActive = true`로 초기화.
    *   **[이어하기]**: `useRunStore`에 `isActive = true`인 데이터가 있을 시 활성화되어 지난 런을 복구하여 재개.

## 2. 향후 구현 계획 (To-Do)

기존 기능들을 더 매끄럽게 엮어줄 필수 UI(모달 등)들의 뼈대 작업을 다음 순서로 기획/구현합니다. (현재 [도감], [설정] 버튼은 Toast 피드백만 출력되도록 처리되어 있습니다.)

### 🔜 1순위: 게임 진행 필수 모달 (GameOver/GameClear)
*   **GameOverModal (결과창)**
    *   **발동 조건**: 플레이어 체력(`playerHp`) 최하 달성 시 즉각 발동.
    *   **내용**: "YOU DIED" 또는 "VICTORY" 텍스트. 최종 도달 층수, 처치한 적 수, 남은 골드 등 요약 리포트 표시.
    *   **동작**: 모달 내 [타이틀로 돌아가기] 버튼 클릭 시, 백엔드의 `isActive` 상태를 `false`로 바꿔 현재 세이브 구간을 박탈하고(영구적 죽음), `MainMenuView`로 라우팅.

### 🔜 2순위: 도감 및 정보 열람 (Compendium)
*   **CompendiumModal (도감)**
    *   **발동 조건**: 메인 메뉴의 [도감] 탭 혹은 인게임 메뉴 상단에서 진입.
    *   **내용**: 게임에 존재하는 모든 카드 데이터베이스(`CARDS` in `cards.ts`)와 유물 데이터베이스(`RELICS` in `relics.ts`)를 열람할 수 있는 UI.
    *   **구성**: '카드', '유물' 두 개의 탭으로 분리하여 그리드(Grid) 형태 렌더링.

### 🔜 3순위: 환경 설정 (Settings)
*   **SettingsModal (환경 설정)**
    *   **발동 조건**: 메인 메뉴 [설정] 탭 또는 전투 중 상단 `⚙️` 버튼 클릭 시 팝업.
    *   **내용**: 음악 및 효과음 조절 슬라이더.
    *   **동작**: 상태관리 라이브러리(`useAudioStore`)를 호출해 값을 변경하여 현재 재생중인 브라우저 오디오 시스템에 반영.

## 💡 개발 노트 (기타 폴리싱 요소)
*   추후 유저가 직접 제작한 고품질 커스텀 에셋(픽셀 아트 몬스터, 배경, 버튼 UI 패널 등)이 도입되면, 기존 이모지(`🛡️`, `⚔️`)와 임시 DOM 객체들을 이미지 텍스처 단위로 교체하는 렌더링 고도화 작업을 수행해야 합니다.
*   보스(고철 기갑수 브루터스) 격파 후 씬이 전환되지 않는 문제를 보완하여 최종 클리어 리포트 창으로 넘어가게끔 후속 조치 예정입니다.
