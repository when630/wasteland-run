## 유물 트리거 시스템

| 트리거 | 함수명 | 해당 유물 |
|--------|--------|----------|
| 전투 시작 | `onBattleStart` | 탄띠, 배지, 탄피, 붕대, 철모, 함정, 구호물자, 나침반, 대형탄약통, 보철, 폐허부적, 주사위 |
| AP 초기화 | `onBattleReset` | 야광시계, 보스 유물(+1 AP) |
| 매 턴 시작 | `onTurnStart` | 배터리, 장전기, 해골부적, 주입기, HUD, 발톱, 정비키트, 모래시계 |
| 매 턴 종료 | `onTurnEnd` | 자기장코일, 양자코어, 조끼, 영구운동 |
| 카드 사용 | `onCardPlayed` | 합금장갑, 어깨받이, 재생연고, 팔찌, 강화장갑, 납판 |
| 카드 소멸 | `onCardExhausted` | 잔해 수집기 |
| 적 처치 | `onEnemyKilled` | 구급상자(특수), 용접장갑(물리), 탄약자석(전체) |
| 치명 피해 | `onFatalDamage` | 가족사진 |
| 전투 종료 | `onBattleEnd` | 인식표(시작유물) |
| 휴식/이벤트 | `onRestOrEventEnter` | 작전지도, 반응로, 통조림, 기술서, 수리도구 |
| 상시 (패시브) | `passive` | 너클, 소음기, 조준경, 혈압조절기, 광전사문양, 멤버십, 농축회복, 정밀공구 |
| 카드 강화 | `onCardUpgrade` | 정밀 공구 |
