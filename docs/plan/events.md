# 웨이스트랜드 런 이벤트 리스트 (26종 + 시작 1종)

> ★ = 기존 구현 | ☆ = 신규
> 챕터 테마: 1막 오염된 외곽 도시 / 2막 무너진 지하철도 / 3막 거대 기업의 방주

---

## 1. 시작 이벤트 (1종)

### ★ 황무지의 관문 (`evt_starting_wasteland_gate`)
폐허가 된 도시의 경계. 세 갈래 길 앞에 이전 탐험가들의 흔적.
- **[보급품 창고를 뒤진다]** → 골드 100
- **[의료 텐트에서 체력을 보강한다]** → 최대 체력 +15, 체력 완전 회복
- **[수상한 사물함을 연다]** → [일반] 유물 1개

---

## 2. 공통 이벤트 (7종)

> 모든 챕터에서 등장. 런 당 성소 이벤트는 각 1회 제한.

### ☆ 강화 성소 (`evt_shrine_upgrade`)
황무지에 홀로 남겨진 낡은 작업대. 공구들이 아직 쓸 만합니다.
- **[장비를 정비한다]** → 카드 1장 선택 **강화**
- **[떠난다]** → 아무 일 없음

### ☆ 정화 성소 (`evt_shrine_purify`)
아직 타오르는 소각로. 불필요한 것을 태울 수 있을 것 같습니다.
- **[필요 없는 장비를 태운다]** → 카드 1장 선택 **제거**
- **[떠난다]** → 아무 일 없음

### ☆ 변환 성소 (`evt_shrine_transform`)
기이한 방사능 결정체가 빛나고 있습니다.
- **[장비를 결정체에 댄다]** → 카드 1장 제거 → 같은 타입 무작위 카드 1장 획득 (**변환**)
- **[떠난다]** → 아무 일 없음

### ★ 떠돌이 기계공의 모닥불 (`evt_mechanic_fire`)
기계팔을 단 기계공이 개조를 제안합니다.
- **[개조 의뢰]** → 골드 50 지불, 카드 1장 **제거** (조건: 골드 50+)
- **[도박성 융합]** → 카드 1장 무작위 제거, [변화] 카드 1장 획득
- **[공격한다]** → 엘리트 전투

### ★ 폐허가 된 진료소 (`evt_abandoned_clinic`)
반쯤 부서진 자동 치료기기가 윙윙거립니다.
- **[장치에 몸을 맡긴다]** → 체력 30 회복, [화상] 카드 1장 추가
- **[진열장을 뒤진다]** → [변화] 카드 1장
- **[무시하고 떠난다]** → 아무 일 없음

### ★ 방랑 상인의 비밀 거래 (`evt_wandering_merchant`)
떠돌이 상인이 거래를 제안합니다.
- **[희귀 카드 구매]** → 골드 75, [희귀] 카드 1장 (조건: 골드 75+)
- **[카드 교환]** → 카드 1장 무작위 제거, [특별] 카드 1장 (조건: 덱 5장+)
- **[유물 흥정]** → 골드 40, [일반] 유물 1개 (조건: 골드 40+)

### ★ 홀로그램 일기장 (`evt_hologram_diary`)
작동하는 홀로그램 프로젝터에 전투 기록이 담겨 있습니다.
- **[기록을 분석한다]** → 카드 1장 선택 **강화** (조건: 미강화 카드 존재)
- **[프로젝터를 분해한다]** → 골드 35, [일반] 카드 1장
- **[무시하고 지나간다]** → 아무 일 없음

## 3. 1~2막 한정 (2종)

> 외곽 도시 ~ 지하철도 사이. 지상 탐색 + 잔해 뒤지기 테마.

### ★ 버려진 군용 트럭 (`evt_abandoned_truck`)
함정이 설치된 군용 트럭.
- **[함정 해체 시도]** → 50% 최대 체력 +5 / 50% 체력 -15
- **[바퀴만 빼간다]** → 골드 30
- **[총알로 자물쇠 부순다]** → [특수 공격] 1장 제거, [보스] 유물 1개 (조건: 특수 공격 보유)

### ★ 고철 수집상의 시체 (`evt_scavenger_corpse`)
무장한 수집상의 시체와 묵직한 배낭.
- **[배낭을 통째로 턴다]** → 체력 -10, 골드 50, 유물(보스 제외) 1개
- **[겉만 뒤져본다]** → 골드 20

## 4. 2~3막 한정 (2종)

> 지하 ~ 방주 사이. 고기술 NPC 조우 테마.

### ★ 기계 사이보그의 내기 (`evt_cyborg_wager`)
사이보그가 피를 걸고 도박을 제안합니다.
- **[동의한다]** → 체력 -15, 33% (희귀 유물 / 골드 100 / 카드 제거) (조건: HP 15+)
- **[기계를 파괴한다]** → [물리 공격] 1장, 50% 엘리트 전투
- **[무시한다]** → 아무 일 없음

### ☆ 방랑 의사 (`evt_wandering_doctor`)
백의를 걸친 노인이 치료를 제안합니다.
- **[치료를 받는다]** → 골드 30, 체력 완전 회복 (조건: 골드 30+)
- **[최대 체력 증강]** → 골드 60, 최대 체력 +10 (조건: 골드 60+)
- **[대가 없이 떠난다]** → 체력 15 회복 (의사의 선의)

## 5. 1막 한정 (5종)

> 오염된 외곽 도시. 방사능, 폐허, 황무지, 모래폭풍 테마.

### ★ 방사능 웅덩이 속의 반짝임 (`evt_radiation_pool`)
형광 녹색 웅덩이 안에 보급 상자가 반짝입니다.
- **[손을 집어넣는다]** → 체력 -10, [희귀] 유물 1개
- **[막대기로 긁어모은다]** → [특수 공격] 카드 1장
- **[무시하고 지나간다]** → 아무 일 없음

### ★ 오염된 오아시스 (`evt_contaminated_oasis`)
방사능 의심 물 웅덩이.
- **[벌컥벌컥 마신다]** → 체력 25 회복, 50% [화상] 2장 추가
- **[정화해서 조금만 마신다]** → 체력 10 회복
- **[변이 식물을 채집한다]** → [특수 방어] 카드 1장

### ★ 작동하는 아케이드 게임기 (`evt_arcade_machine`)
폐허 건물에서 기적적으로 전력이 들어오는 오락기.
- **[플레이한다]** → 체력 -5, 카드 2장 무작위 **강화** (조건: HP 5+)
- **[부숴버린다]** → [물리 방어] 카드 1장

### ★ 무너진 방공호 (`evt_collapsed_shelter`)
폭격으로 반쯤 무너진 방공호.
- **[깊이 들어간다]** → 40% [희귀] 카드 + 골드 50 / 60% 체력 -20 + [화상]
- **[입구만 뒤진다]** → 골드 25, 체력 10 회복
- **[무전기로 도움 요청]** → 50% 유물 / 50% 무응답

### ☆ 모래폭풍 속 대피소 (`evt_storm_shelter`)
모래폭풍이 몰아칩니다. 근처에 작은 동굴이 보입니다.
- **[동굴에서 쉰다]** → 체력 20 회복
- **[폭풍 속을 뒤진다]** → 체력 -8, 유물(보스 제외) 1개
- **[폭풍을 그냥 맞는다]** → 체력 -5, 다음 전투 첫 턴 AP +2

## 6. 2막 한정 (5종)

> 무너진 지하철도. 터널, 변이 생물, 암시장, 지하 유적 테마.

### ★ 변이체 거래상 (`evt_mutant_trader`)
세 개의 눈을 가진 변이체가 거래를 제안합니다.
- **[최대 체력을 판다]** → 최대 체력 -8, [희귀] 유물 1개 (조건: 최대 HP 8+)
- **[카드를 2장 제거한다]** → 카드 2장 제거, 최대 체력 +15
- **[공격한다]** → 엘리트 전투

### ☆ 잊혀진 무기고 (`evt_forgotten_armory`)
지하 군사 시설의 무기 보관함. 세 가지 무기가 남아있지만 하나만 들 수 있습니다.
- **[근접 무기를 집는다]** → [희귀] 물리 공격 카드 1장, 체력 -5
- **[화기를 집는다]** → [희귀] 특수 공격 카드 1장, 체력 -5
- **[방어구를 집는다]** → [희귀] 방어 카드(물리/특수 랜덤) 1장, 체력 -5

### ☆ 고철 도박사 (`evt_scrap_gambler`)
지하 시장에서 고철 룰렛을 돌리는 도박사.
- **[소액 베팅 (골드 25)]** → 50% 골드 50 / 50% 골드 잃음 (조건: 골드 25+)
- **[대형 베팅 (골드 50)]** → 40% [희귀] 유물 / 30% 골드 100 / 30% 골드 잃음 (조건: 골드 50+)
- **[도박사를 턴다]** → 골드 20, 50% 엘리트 전투

### ☆ 지하철 유령 열차 (`evt_ghost_train`)
어둠 속에서 헤드라이트를 켠 열차가 천천히 다가옵니다. 무인 자동 운행 중인 것 같습니다.
- **[열차에 올라탄다]** → 50% 골드 80 + 카드 1장 강화 / 50% 체력 -12 + [화상] 1장
- **[선로를 뒤진다]** → 골드 30, 탄약 관련 [변화] 카드 1장
- **[지나가기를 기다린다]** → 아무 일 없음

### ☆ 터널 속 은신처 (`evt_tunnel_hideout`)
누군가가 터널 옆방에 만들어둔 은신처. 침낭과 보급품이 있습니다.
- **[하룻밤 머문다]** → 체력 25 회복, 카드 1장 선택 **강화**
- **[보급품을 챙긴다]** → 골드 40, [일반] 카드 1장
- **[함정이 있을 수 있다]** → 체력 -5, [특별] 유물 1개

## 7. 3막 한정 (4종)

> 거대 기업의 방주. AI, 첨단 기술, 기업 실험, 보안 시스템 테마.

### ☆ 고대 복제 장치 (`evt_ancient_duplicator`)
구시대 기업의 실험용 복제기가 에너지를 뿜고 있습니다.
- **[카드를 복제한다]** → 카드 1장 선택, 동일 카드 **복제** 추가
- **[장치를 분해한다]** → 골드 60
- **[떠난다]** → 아무 일 없음

### ☆ AI 진단 시스템 (`evt_ai_diagnostic`)
기업 의료 AI가 아직 가동 중입니다. "스캔을 시작합니다. 최적화 프로토콜을 실행하시겠습니까?"
- **[풀 스캔]** → 카드 2장 **강화**, 덱에 [방사능 오염] 1장 추가
- **[퀵 스캔]** → 카드 1장 **강화**
- **[해킹 시도]** → 체력 -10, [희귀] 카드 1장

### ☆ 기업 실험실 (`evt_corporate_lab`)
방주 내부의 생체 실험실. 다양한 실험 약품이 진열되어 있습니다.
- **[혈청을 주입한다]** → 최대 체력 +8, [화상] 1장 추가
- **[안정제를 복용한다]** → 덱의 모든 상태이상 카드 **제거**
- **[약품을 파괴한다]** → 골드 50, 무작위 [변화] 카드 1장

### ☆ 최종 보안 프로토콜 (`evt_security_protocol`)
방주의 보안 시스템이 침입자를 감지했습니다. 잠긴 문 너머에 귀중한 장비가 보입니다.
- **[해킹한다]** → 체력 -8, [희귀] 유물 1개
- **[우회한다]** → 골드 30, 카드 1장 **강화**
- **[경보를 울린다]** → 엘리트 전투, 승리 시 [희귀] 유물 + 골드 80

## 이벤트 등장 규칙

| 규칙 | 설명 |
|------|------|
| **챕터 필터** | 각 이벤트는 지정된 챕터에서만 등장 |
| **성소 1회 제한** | 강화/정화/변환 성소는 런 당 각 1회 |
| **중복 방지** | 동일 런에서 같은 이벤트 재등장 불가 |
| **골드/덱 조건** | 특정 선택지는 골드/카드 조건 충족 시에만 활성화 |
| **가중치** | 성소: 낮음, 일반: 동일, 챕터 전용: 해당 챕터에서 높음 |
