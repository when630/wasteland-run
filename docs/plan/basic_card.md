# ⛺ 기본 시작 덱 (10장) 및 강화(Upgrade) 상세 스펙

**기본 조건:** 매 전투 시작 시 3 AP / 0 탄약으로 시작합니다.

---

### 1. [물리 공격] 낡은 쇠파이프 (보유량: 4장)
* **기본:** 1 AP 소모 / **6** 물리 피해
* **강화 (낡은 쇠파이프+):** 1 AP 소모 / **9** 물리 피해 (+3)
* **기획 의도:** 초반 챕터를 뚫기 위해 가장 무난하고 필수적인 타격치 증가입니다. 1AP당 가치가 확연히 올라가므로 초반 딜 부족을 해결해 줍니다.

### 2. [물리 방어] 엄폐 (보유량: 3장)
* **기본:** 1 AP 소모 / **5** 물리 방어도
* **강화 (엄폐+):** 1 AP 소모 / **8** 물리 방어도 (+3)
* **기획 의도:** 초반 잡몹들이 주로 가하는 7~8 언저리의 물리 피해를 카드 한 장으로 온전히 막아낼 수 있게 하여 플레이어의 체력 관리에 큰 도움을 줍니다.

### 3. [특수 방어] 젖은 천 두르기 (보유량: 1장)
* **기본:** 1 AP 소모 / **5** 특수 방어도
* **강화 (젖은 천 두르기+):** 1 AP 소모 / **8** 특수 방어도 (+3)
* **기획 의도:** 물리 방어와 동일한 성장치를 가집니다. 1챕터 엘리트나 보스가 강력한 특수 공격(독 브레스, 화염병 등)을 사용할 때 생사를 가르는 튜토리얼 성격의 핵심 방어 카드입니다.

### 4. [변화] 잔해 뒤지기 (보유량: 1장)
* **기본:** **1 AP** 소모 / 1 탄약 획득
* **강화 (잔해 뒤지기+):** **0 AP** 소모 / 1 탄약 획득 (비용 감소)
* **기획 의도:** 수치 증가가 아닌 **'코스트 감소'**가 핵심인 전략적 강화입니다. 0 코스트가 되면 턴의 행동력(3 AP)을 온전히 공격과 방어에 쏟아부으면서도 탄약을 장전할 수 있어 전투의 템포와 덱 사이클이 비약적으로 매끄러워집니다.

### 5. [특수 공격] 녹슨 권총 (보유량: 1장)
* **기본:** 1 AP + 1 탄약 소모 / **10** 특수 피해
* **강화 (녹슨 권총+):** 1 AP + 1 탄약 소모 / **14** 특수 피해 (+4)
* **기획 의도:** 제한된 특수 자원인 '탄약'을 소모하는 만큼 타격 카드보다 데미지 상승 폭을 더 높게 설정(+4)했습니다. 물리 방어도가 높은 적들을 한 방에 관통하는 강력한 조커 카드로 활용됩니다.
