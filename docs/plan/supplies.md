# 보급품 (Emergency Supplies) 시스템 설계

## 기본 규칙

- **컨셉**: 황무지에서 수집한 비상 보급품 — 식량, 의약품, 군용 장비, 임시 장치
- **소지 한도**: 기본 3개 (유물로 확장 가능)
- **사용 타이밍**: 전투 중 자기 턴에 무료 액션으로 사용 (AP 소모 없음)
- **획득 경로**: 전투 보상 / 상점 구매 / 이벤트 / 보물방
- 일부 `🗺️` 표시 항목은 비전투(맵)에서도 사용 가능

## 밸런스 가이드

### 상점 가격
- COMMON: 50~75G
- UNCOMMON: 100~150G
- RARE: 200~300G

### 전투 보상 드롭률
- 일반전: 30% 확률로 COMMON 1개
- 엘리트전: 50% 확률로 UNCOMMON 1개
- 보스전: 100% RARE 1개

### 설계 의도
- 소지 한도 3개가 핵심 — "언제 쓸지" 판단이 전략 요소
- COMMON은 단순 수치 보조, RARE는 전투 흐름을 뒤집는 수준
- 금욕의 서약은 보급품 의존도 높은 플레이어에게 큰 트레이드오프
- 비전투 사용 가능 항목(🗺️)은 회복/강화 계열로 제한하여 남용 방지
