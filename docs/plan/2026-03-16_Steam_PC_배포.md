## 남은 작업 (Windows에서 직접 진행)

### 1단계: Electron 동작 확인 (Windows)

```powershell
cd frontend
npm run electron:dev
```

- [ ] 앱 창이 정상적으로 열리는지 확인
- [ ] Pixi.js WebGL 전투 캔버스 정상 렌더링 확인
- [ ] ESC 키 → 설정 모달 동작 확인
- [ ] F11 풀스크린 토글 확인
- [ ] 새 게임 → 세이브 → 앱 종료 → 재실행 → 이어하기 동작 확인
- [ ] 볼륨 설정 변경 후 재실행 시 복원 확인
- [ ] 통계 누적 확인 (런 종료 후 통계 페이지)

### 2단계: 프로덕션 빌드 (Windows)

```powershell
cd frontend
npm run electron:build
```

- [ ] `release/` 디렉토리에 NSIS 인스톨러 생성 확인
- [ ] 인스톨러 실행하여 설치 → 앱 실행 확인
- [ ] 디버그 메뉴가 숨겨져 있는지 확인

### 3단계: 앱 아이콘 제작

- [ ] 512x512 PNG 아이콘 원본 제작
- [ ] `resources/icon.ico` (Windows) 변환
- [ ] `resources/icon.icns` (macOS, 선택사항) 변환
- [ ] `resources/icon.png` (Linux) 배치
- [ ] `electron-builder.yml`에서 아이콘 경로 확인

### 4단계: Steam SDK 연동 (Phase 2)

#### 사전 준비
- [ ] Steamworks 파트너 계정 등록 (partner.steamgames.com)
- [ ] Steam App ID 발급
- [ ] `steam_appid.txt`에 실제 App ID 입력

#### 코드 작업
```bash
cd frontend
npm install steamworks.js
```

- [ ] `electron/steam.ts` 주석 해제 및 실제 App ID 연결
- [ ] `electron/main.ts`에서 `initSteam()` 호출 추가
- [ ] `platform.ts`의 `platformSubmitLeaderboard` / `platformLoadLeaderboard` → Steam API 연결
- [ ] 업적 트리거 연결:
  - 첫 전투 승리 → `FIRST_VICTORY`
  - 챕터 1/2/3 클리어 → `CHAPTER1/2/3_CLEAR`
  - 누적 처치 100/500 → `KILL_100/500`
  - 누적 골드 1000 → `GOLD_1000`
  - 보스전 무피해 → `NO_DAMAGE_BOSS`
  - 유물 5개 이상 → `FULL_RELIC`
- [ ] Steam 클라우드 세이브 설정 (Steamworks 웹 대시보드에서 디렉토리 지정)

### 5단계: Steam 배포

- [ ] Steamworks 대시보드에서 디포(Depot) 생성
- [ ] steamcmd로 빌드 업로드
- [ ] 스토어 페이지 작성 (스크린샷, 설명, 태그)
- [ ] Steam 오버레이 동작 테스트
- [ ] 업적 동작 테스트
- [ ] 클라우드 세이브 동기화 테스트
- [ ] 리더보드 동작 테스트
- [ ] 내부 QA 완료 후 스토어 게시

## 현재 프로젝트 구조

```
wasteland-run/
  frontend/
    electron/                  # Electron 메인 프로세스
      main.ts                  # BrowserWindow, IPC, F11, 창 상태
      preload.ts               # contextBridge IPC 브릿지
      storage.ts               # userData JSON 저장/로드
      steam.ts                 # Steam SDK (Phase 2)
    src/
      api/
        platform.ts            # Electron IPC 래퍼
      store/                   # Zustand 스토어
      components/              # React UI + Pixi.js
      logic/                   # 순수 게임 로직
      ...
    electron.vite.config.ts    # electron-vite 빌드 설정
    electron-builder.yml       # 패키징 설정
    steam_appid.txt            # Steam App ID
    resources/                 # 앱 아이콘 (제작 필요)
  docs/
    plan/                      # 기획 문서
    logs/                      # 작업 로그
```
