# 2026-03-17 Galmuri11 폰트, 맵 메커니즘, 유물 전체 구현

## 작업 요약

### 1. Galmuri11 한글 픽셀 폰트 적용
- Galmuri11 Regular/Bold woff2 번들 추가 (`src/assets/fonts/`)
- `index.css`에 `@font-face` 등록 + 전역 `font-family` 설정

### 2. 맵 메커니즘 구현
- **EVENT 노드 동적 인카운터 결정**: 진입 시 적/상인/보물/이벤트 확률 기반 동적 결정
  - 확률 변동 규칙 (기본 확률 초기화/증가, 6층 이후 보정, 오버플로 보정, 상인 제외)
  - `useRunStore`에 `unknownProb`, `unknownVisitedCount`, `lastVisitedNodeType` 추가
- **보물방**: 3개 선택 → 랜덤 1개 상자 (소형 50%, 중형 33%, 대형 17%)
- **상점 카드 제거**: 고정 50G → 75G + 25G×횟수 누적
- **전투 보상 카드 등급 확률**: 일반(60/30/10), 엘리트(50/30/20), 보스(희귀 100%)
- **골드 보상 랜덤 범위**: 일반 10~20, 엘리트 25~35, 보스 90~110
- **이벤트 챕터 필터링**: `chapters` 필드 적용, `oncePerRun` 중복 방지
- **보물방 뱃지** 에셋 추가 (relic_badge.webp)
- map.md 기획서 전면 개선
