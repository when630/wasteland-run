## 4. 이벤트 2종 추가

기존 7개 이벤트에 2개를 추가하여 총 9개 랜덤 이벤트로 확장했습니다.

### 4.1 방랑 상인의 비밀 거래 (`evt_wandering_merchant`)
* **선택지 A**: 골드 75 → 무작위 [희귀] 카드 1장 구매
* **선택지 B**: 덱 카드 1장 무작위 제거 → 무작위 [비범한] 카드 1장 획득
* **선택지 C**: 골드 40 → 무작위 [일반] 유물 획득

### 4.2 오염된 오아시스 (`evt_contaminated_oasis`)
* **선택지 A**: 체력 25 회복 + 50% 확률로 [화상] 카드 2장 추가
* **선택지 B**: 체력 10 회복 (안전)
* **선택지 C**: 무작위 [특수 방어] 카드 1장 획득

### 구현 위치

## 6. 게임오버 모달 개선

기존 단순 통계 나열에서 시각적으로 개선된 결과 화면으로 재구성했습니다.

### 추가된 요소
* **클리어 등급 (S/A/B/C)**: 시간, 체력, 처치 수 기반 점수 계산
  * S등급: 85점 이상 (빠른 클리어 + 높은 체력 + 많은 처치)
  * A등급: 70점 이상, B등급: 50점 이상, C등급: 그 외
* **획득 유물 목록**: 이미지 아이콘으로 표시, 호버 시 이름/설명 툴팁
* **최종 덱 요약**: 카드 타입별 보유 수량 표시
* **2열 레이아웃**: 좌측 통계 + 우측 유물/덱 정보로 정보 밀도 향상

### 구현 위치
* `frontend/src/components/ui/GameOverModal.tsx` — 전면 재작성

## 7. 기타 UI/UX 개선 (이전 세션에서 완료)

이번 리포트 작성 시점 기준, 같은 날 앞서 진행한 작업들도 포함합니다.

### 7.1 맵 오버레이 시스템
* HUD에 🗺️ 지도 버튼 추가
* 전투/상점/이벤트/모닥불 등 모든 씬에서 오버레이로 맵 확인 가능
* `MapView`에 `viewOnly` + `onClose` props 추가

### 7.2 맵 시각 개선
* 양피지 스타일 배경 (parchment gradient)
* 뱃지 이미지만 사용하는 노드 렌더링
* 노드 위치 랜덤 지터 (자연스러운 지도 느낌)
* 방문한 노드 활성 상태 유지

### 7.3 플레이어 버프/디버프 표시
* Pixi.js 캔버스에 플레이어 하단 아이콘 표시 영역 추가
* 무료 공격, 물리 불가, 카드 보존, 반사, AP 보너스, 탄약 보너스, 낙인, 파워, 취약, 약화 등 전체 상태 표시

### 7.4 카드 UI 개선
* 무료 물리 공격 시 AP 코스트를 0으로 표시 (녹색 뱃지)
* 물리 공격 불가 시 카드 잠금 처리 (반투명 + 빨간 테두리 + 🔒 오버레이)

### 7.5 적 시스템 리팩토링
* `EnemyTier` 타입 추가 (`NORMAL` | `ELITE` | `BOSS`)
* 모든 적에 `tier` 필드 부여
* `getEnemyIdsByTier()` 헬퍼 함수로 씬별 소환 분기 일원화
* 보스 판정을 `baseId` 하드코딩 → `tier === 'BOSS'`로 전환

### 7.6 타겟팅 곡선 조정
* SVG Quadratic Bezier 곡선의 X 오프셋을 +100 → -150, Y를 -150 → -200으로 변경
* 왼쪽으로 더 아치형의 곡률 생성

### 7.7 유물 이미지 매칭 수정
* 8개 유물 이미지 파일을 실제 이미지 내용에 맞게 물리적으로 파일명 변경 (circular rename)
