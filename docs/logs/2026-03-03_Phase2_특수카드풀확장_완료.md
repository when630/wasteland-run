# 🚀 Phase 2 - Week 5: 탄약(Ammo) 시너지 덱 업데이트 완료

## 1. 개요
* **날짜:** 2026-03-03
* **작업 내용:** 기본 시작 덱에 특수 자원 획득-소모 메커니즘을 부여하는 신규 카드 2종 편입 및 시스템 확장.

## 2. 작업 상세 내용
* **신규 데이터 추가 (`assets/data/cards.ts`):** 
  * `rusty_pistol` (녹슨 권총): `1 AP`, `1 Ammo`를 모조리 소모하여 무려 15피해를 가하는 고화력 특수 공격.
  * `scavenge` (잔해 뒤지기): `1 AP`만 소모하고, 게임 고유 자원인 `Ammo`를 2 획득하는 유틸리티 변화.
  * 위 카드 2장을 플레이어 초기 덱 10장에 1장씩 교체 배분 완료.
* **이펙트 컴파일러 확장 (`useCardPlay.ts`):**
  * 카드 효과 오브젝트 처리기에서 `type === 'ADD_AMMO'`를 인식하여 즉각적으로 HUD 패널(`useBattleStore.addAmmo`)의 수치를 올리도록 연동 배선 구축.

## 3. 남은 단상
* 이제 초기 턴만 돌려도 물리 공격을 퍼부을지, 아니면 방어를 포기하는 대신 권총을 장전(`잔해 뒤지기` -> `녹슨 권총` 콤보)할지 선택의 기로에 놓일 수 있습니다.
* 탄약 자원 처리가 완성되었으니 차후 산성 액체 살포(산성 중첩, 화상) 기믹이나 드로우 로직이 들어간 신규 카드 제작이 용이합니다. 다음 주차(Week 6)인 픽셀 아트 애니메이션 및 시각적 연출(폴리싱) 파트로 전환하여 화면을 게임답게 다듬어야 합니다.
